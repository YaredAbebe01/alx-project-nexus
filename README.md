## RideWave

RideWave is a ride-hailing web app UI that lets users book a ride, track a driver, and view trip history. It includes a mock auth flow, driver selection, and a map preview.

## Tech Stack

- Next.js 16 (App Router)
- React 19 + TypeScript
- Tailwind CSS + shadcn/ui (Radix UI)
- next-themes for light/dark mode

## Features

- Mock authentication (login/signup) with profile info
- Ride booking flow with driver selection and status steps
- Trips history and profile pages
- Light/dark theme toggle
- Static map preview for Addis Ababa

## How to Run

Install dependencies:

```bash
npm install
```

If you hit peer dependency errors, use:

```bash
npm install --legacy-peer-deps
```

Start the dev server:

```bash
npm run dev
```

Then open http://localhost:3000
